# Книжный каталог версия 2.0

Простое веб-приложение для учёта книг. Добавляй, удаляй, редактируй и листай по страницам.

## Что умеет

- Добавление автора (имя, у каждого автора уникальный ID)
- Добавление книги (название, уникальный ID автора)
- Список всех книг с пагинацией (6 книг на страницу)
- Прямой ввод номера страницы (выбираешь номер страницы собственноручно)
- Список всех авторов (имя, уникальный ID)
- Поиск автора по ID в списке авторов
- Удаление книги (кнопка рядом с каждой)
- Редактирование названия книги

## Чем лучше прошлой версии

- Улучшенная пагинация (Добавлены стрелки назад и вперед, также работает без ошибок)
- Добавлены разные проверки (К примеру на наличие книг, условно если список книг пустой то кнопки пагинации выключаются)
- Добавление книг с проверкой (Если ID в поле ввода уникального ID автора не существует, книгу нельзя будет добавить)
- Две таблицы с Include (INNER JOIN) (книги и авторы связаны через AuthorId, данные объединяются на сервере)

## Как запустить

1. Установи .NET 8 (если ещё нет)
2. Скопируй репозиторий или скачай zip
3. Открой терминал в папке с проектом
4. Введи `dotnet run`
5. Открой браузер и перейдите на `http://localhost...` (В консоли будет порт)

## Технологии

- C# и ASP.NET Core (Minimal API)
- Entity Framework Core + SQLite
- HTML, CSS, JavaScript (fetch, DOM)

## Пара моментов

- База данных создаётся автоматически при первом запуске (файл bookslist.db)
- Если меняешь модель, удали старый .db файл, чтобы EF Core создал таблицы заново
- Навигация:
1. index.html (главная)
2. addBook.html / addAuthor.html (Добавление книги / Автора)
3. bookList.html / authorList.html (Список книг / Авторов)
